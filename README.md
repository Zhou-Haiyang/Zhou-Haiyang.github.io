# Minimal academic website

This folder is the deployable website. Copy its contents, not the folder itself, to the root of the GitHub repository named `Zhou-Haiyang.github.io`.

Configured username: `Zhou-Haiyang`.

Validate from the package root:

```bash
python scripts/validate_site.py starter_site
```

Preview locally:

```bash
python -m http.server 8000 --directory starter_site
```
